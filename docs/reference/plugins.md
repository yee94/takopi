# Plugins

Community and third-party plugins that extend yee88.

| Plugin | Type | Description |
|--------|------|-------------|
| [yee88-matrix](https://github.com/Zorro909/yee88-matrix) | Transport | Matrix protocol backend with E2EE, voice transcription, and multi-room support |
| [yee88-scripts](https://github.com/asianviking/yee88-scripts) | Command | Dynamic script runner for executing Python scripts via `/run` command |
| [yee88-discord](https://github.com/asianviking/yee88-discord) | Transport | Discord bot backend for interacting with yee88 via Discord |
| [yee88-slack](https://github.com/richardliang/yee88-slack-plugin) | Transport | Slack bot backend for interacting with yee88 via Slack |
